# Производительность LDUP v5 (CPU Benchmark)

## 1. Целевые показатели (KPK)

| Операция | Цель (v5) | Обоснование |
| :--- | :--- | :--- |
| **Индексация 44-ФЗ** | 30 сек | Потоковое чтение + FTS5. |
| **Генерация C³** | 100 ст./мин | Параллельные API-вызовы. |
| **Поиск (Retrieval)** | < 200 мс | Локальный BM25 + ONNX. |
| **RAM (Peak)** | < 512 МБ | Стриминг WordML. |

## 2. Оптимизация для CPU
1.  **SIMD (Single Instruction Multiple Data):** Использование `fastembed` с библиотекой `intel-extension-for-pytorch` (IPEX) или чистым ONNX для ускорения векторизации.
2.  **Thread-safe SQLite:** Использование режима WAL (Write-Ahead Logging) для параллельного поиска во время индексации.
3.  **Lazy Extraction:** Не извлекать таблицы и картинки, если они не требуются для семантического анализа (экономия 30% ресурсов парсинга).
