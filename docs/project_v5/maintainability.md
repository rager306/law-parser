# Поддержка и развитие LDUP v5

## 1. uv Workspace
Проект организован как `uv workspace` с разделением на подпакеты:
*   `ldup-ingester`
*   `ldup-core`
*   `ldup-api`

Это позволяет обновлять парсер XML отдельно от логики LLM.

## 2. Самообучение (Self-Improvement)
Интеграция с инструментом `rlm_analyze` позволяет автоматически находить новые паттерны изменения закона и предлагать новые `StructuralRule`.

## 3. Документирование
*   Автоматическая генерация документации через `mkdocs` на основе докстрингов и Pydantic-схем.
*   Визуализация графа связей через Mermaid в реальном времени.
